# VrcDmaFish 🎣

专业的 VRChat DMA 全自动钓鱼机器人。支持硬件级降维打击，极致安全。

## 🚀 快速开始
1. **克隆仓库**: `git clone https://github.com/Naomi-Armitage/vrc-dma-fish`
2. **环境配置**: 右键运行 `setup.ps1` (自动处理海外/国内网络环境，部署所有依赖)。
3. **编译运行**: `dotnet run`

## 🛠️ 配置说明
启动后将进入 **交互式 UI 向导**，您可以在界面中配置：
- KMBOX 端口/IP
- 钓鱼张力阈值 (自动保存至 appsettings.json)
